# WebSocket Project

This project demonstrates a simple WebSocket server and client implemented 
in Python, and follows multiple tutorials from the following places:

* [Building Real Time Applications With Python and WebSockets](https://medium.com/@AlexanderObregon/building-real-time-applications-with-python-and-websockets-eb33a4098e02) by @AlexanderObregon on Medium.com
* [html5 websocket connecting to python](https://stackoverflow.com/questions/18240358/html5-websocket-connecting-to-python) by gen, mccakici on StackOverflow.com

## Prerequisites

* Python 3.9 or higher (probably)
* pip
